基于Hexo框架的博客搭建

发表于 2019-03-15

说明

win10系统下基于Github和Hexo搭建个人博客

环境配置

1.基本环境

win10，Git，node.js，Hexo框架

2.GitHub创建个人仓库

登录GitHub，创建新的仓库，名称格式为：用户名.github.io

3.安装Git

下载路径：Git下载

根据需要选择路径进行安装

4.绑定GitHub

在Git安装完成后，打开任意文件夹，在空白处右键，选择Git Bash Here
在打开的命令行中输入：

git config –global “name”

git config –global “email”

注意：用户名和注册邮箱根据自己实际情况填写

5.设置免密钥登录：

在上述命令行输入：
ssh-keygen -t rsa -C “email”，然后连续三个回车生成密钥

打开C:\Users.ssh\id_rsa.pub文件(提示：可用记事本格式打开)，复制全部内容，打开页面，选择new SSH Key，将刚才复制的内容全部粘贴到Key文本框中，Title中任意填写，最后点击Add SSH key

检测：
在Git Bash中输入 ssh git@github.com，如果显示：

6.安装Node.js

下载：Node下载

安装：根据需要在合适的路径安装(安装会自动添加环境变量)

检测：在Git Bash命令行中输入node -v或者node –version，显示

输入npm -v或者npm –version，显示

7.安装Hexo

在合适的位置创建文件夹，如：D:\Blog，然后进入Blog，打开命令行，输入npm install -g hexo-cli
如果安装失败，则修改目标源，输入npm config set registry https://registry.npm.taobao.org
